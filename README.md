# analise-exploratoria-covid19
- Análise Exploratória dos dados da Síndrome Gripal disponíveis através da Plataforma [OpenDataSUS](https://opendatasus.saude.gov.br/dataset/casos-nacionais). Os dados utilizados foram do Estado do Pará, na qual pode ser encontrado no link anterior. Neste repositório foi adicionado a documentação dos dados, que descreve cada coluna, juntamente com as suas possíveis informações.
## Ferramentas utilizadas
1. Google Colab
2. Biblioteca Pandas
